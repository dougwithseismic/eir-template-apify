# Apify for Entrepreneurs: The Marketplace Arbitrage Play

Welcome to the entrepreneur's guide to building profitable Apify actors without building a full product from scratch. This guide is specifically designed for the **Entrepreneurs in Residence** program, showing you how to leverage existing marketplaces to create value quickly and start generating revenue.

## Why Build on Marketplaces?

**The Core Insight:** You don't need to build an entire product, acquire customers, or create a brand to start making money. Marketplaces like Apify already have:

- **Built-in traffic** - Millions of monthly visitors searching for solutions
- **Payment infrastructure** - Billing, subscriptions, and payments handled
- **Discovery mechanisms** - SEO, search, and recommendations drive customers to you
- **Trust and credibility** - Marketplace brand gives you instant legitimacy
- **Support infrastructure** - Platform handles basic customer support and infrastructure

**Your Job:** Build something valuable that solves a real problem, and let the marketplace do the heavy lifting.

#### Building Your First Actor

**Step 1: Configure Your Actor**

Edit `apps/actor/.actor/actor.json`:
```json
{
  "actorSpecification": 1,
  "name": "your-niche-scraper",
  "title": "Your Niche Scraper - Descriptive Title",
  "description": "Extract [specific data] from [platform] with ease. Built for [target audience].",
  "version": "1.0.0",
  "buildTag": "latest",
  "environmentVariables": {},
  "dockerfile": "./Dockerfile"
}
```

**Step 2: Define Input Schema**

Edit `apps/actor/.actor/INPUT_SCHEMA.json`:
```json
{
  "title": "Your Actor Input",
  "type": "object",
  "schemaVersion": 1,
  "properties": {
    "startUrls": {
      "title": "Start URLs",
      "type": "array",
      "description": "URLs to scrape",
      "editor": "requestListSources",
      "example": [
        { "url": "https://example.com/page" }
      ]
    },
    "maxResults": {
      "title": "Max Results",
      "type": "integer",
      "description": "Maximum number of results to scrape",
      "default": 100,
      "minimum": 1
    }
  },
  "required": ["startUrls"]
}
```

**Step 3: Implement Your Scraper**

This template includes **Playwright** for browser automation and **Claude Code** to help you write the scraper:

```typescript
// apps/actor/src/main.ts
import { Actor } from 'apify';
import { PlaywrightCrawler } from 'crawlee';

await Actor.init();

const input = await Actor.getInput();
const { startUrls, maxResults = 100 } = input;

const crawler = new PlaywrightCrawler({
    maxRequestsPerCrawl: maxResults,
    async requestHandler({ request, page, enqueueLinks, log }) {
        log.info(`Processing: ${request.url}`);

        // YOUR SCRAPING LOGIC HERE
        // Use Playwright to interact with the page
        // Extract the data you need

        const data = {
            url: request.url,
            title: await page.title(),
            // ... extract more data
        };

        // Save the results
        await Actor.pushData(data);
    },
});

await crawler.run(startUrls.map(url => ({ url: url.url })));
await Actor.exit();
```

## Frequently Asked Questions

**Q: Do I need to be a developer?**
A: Basic JavaScript/TypeScript knowledge helps, but this template + Claude Code makes it much easier. Many entrepreneurs learn as they build.

**Q: How long until I make money?**
A: Realistically: 4-8 weeks for first revenue. 3-6 months to $1K+/month. Depends on niche and execution.

**Q: What if my actor doesn't sell?**
A: Build multiple actors (portfolio approach). Learn from each one. Not every actor will be a hit, but the lessons compound.

**Q: How much can I really make?**
A: Top actors: $10K-50K/month. Solid actors: $1K-10K/month. Niche actors: $200-2K/month. Most entrepreneurs build portfolios earning $3K-15K/month total.

**Q: Is this legal?**
A: Scraping publicly available data is generally legal. Always check robots.txt and terms of service. Focus on business data, not personal data.

**Q: How much does it cost to run an actor?**
A: Apify charges for compute time. Efficient actors cost pennies per run. You set your pricing to cover costs + profit.

**Q: Can I do this part-time?**
A: Yes! Many entrepreneurs build actors on nights/weekends. Initial build: 20-40 hours. Maintenance: 2-5 hours/week.

**Q: What about competition?**
A: There's room for multiple actors in each niche. Differentiate through better docs, support, features, or price. Focus on underserved niches.
